What is the purpose of this API -

The xkcd API lets you fetch comics and their metadata which includes the date it appeared, its title, a transcript, alternate text, and the image source file for visual output.

Getting Started with this API -

First, you should be aware that to access this API, another API is required. In this case CORSANYWHERE - Cross-Origin Resource Sharing. Most of the API service providers will have CORS enabled in their response object as they are meant to be consumed by different domains.

The link to the API's homepage -

http://xkcd.com/

What do you need to do to be able to use this API (OAuth, APIKey, completely open)? -

The API is open. While the API is open, you are able to obtain a Key by contacting the API author for private usage. CORS is required.
